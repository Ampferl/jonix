# Jonix
> Jonix is a simple kernel written in C++. 

## Features
- PS/2 Keyboard and Mouse support
- PCI driver
- ATA Driver
- Malloc and free
- Paging
- Scheduling
- Simple GUI
- VGA Mode
- Own LibC implementation
- Custom toolchain
